## Descripción del Programa

Este proyecto es una aplicación de consola desarrollada en Python que permite gestionar información sobre países. El sistema lee los datos desde un archivo CSV y los almacena en memoria utilizando listas y diccionarios para facilitar su manipulación.

A través de un menú interactivo, el usuario puede:

- Agregar nuevos países.
- Actualizar la población y superficie de los países existentes.
- Buscar países por coincidencia exacta o parcial de nombre.
- Aplicar filtros avanzados (por continente, rango de población y rango de superficie).
- Ordenar los datos bajo distintos criterios (nombre, población, superficie).
- Obtener estadísticas clave (países con mayor/menor población, promedios y conteos por continente).

## Instrucciones de Uso

### Requisitos Previos

- Python 3.x instalado en el sistema.
- El archivo `paises.csv` debe estar ubicado en la misma carpeta que el archivo principal `main.py`.

### Ejecución

Para iniciar el programa, abre una terminal o consola, navega hasta la carpeta del proyecto y ejecuta el siguiente comando:

```bash
python main.py
```
